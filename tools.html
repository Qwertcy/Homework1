White-hat Hacking

Network Scanning & Enumeration

Netcat (nc) – A command-line tool for reading, writing, and redirecting data across network connections.
Wireshark – A network protocol analyzer that captures and inspects packets to detect vulnerabilities or network anomalies.


Vulnerability Assessment

Nessus – A vulnerability scanner that detects security holes in networks, systems, and applications.
Nikto – A web server scanner that tests for security vulnerabilities like outdated software and misconfigurations.


Penetration Testing & Exploitation

Metasploit Framework – A powerful penetration testing tool that helps identify, exploit, and verify vulnerabilities.
John the Ripper – A password-cracking tool that brute-forces passwords to test password strength.


Forensics & Incident Response

Autopsy – A digital forensics tool that helps analyze disk images and recover deleted files.
Volatility Framework – A memory forensics tool used to extract digital artifacts from RAM to detect malware or hidden processes.


Malware Analysis

Cuckoo Sandbox – An automated malware analysis tool that runs suspicious files in a virtual environment to detect malicious behavior.
Ghidra – A reverse engineering framework developed by the NSA for analyzing compiled code and malware.


Web & Application Security

ZAP (OWASP Zed Attack Proxy) – A web security scanner that detects vulnerabilities in web applications.
Wappalyzer – A browser extension that identifies technologies used by websites (e.g., CMS, frameworks, analytics tools).


Wireless Security & Network Sniffing

Aircrack-ng – A suite of tools used for Wi-Fi network security auditing, including WEP/WPA cracking.
Kismet – A wireless network detector and intrusion detection tool that can capture network packets.


OSINT (Open-Source Intelligence) & Social Engineering

theHarvester – A tool for gathering email addresses, subdomains, and hostnames from public sources.
Maltego – A data visualization tool that maps relationships between people, companies, domains, and infrastructure.
